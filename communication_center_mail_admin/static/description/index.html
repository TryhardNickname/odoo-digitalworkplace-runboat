Odoo chatter message edit and delete

Avoid Typos by editing and deleting messages in Odoo Chatter!

Odoo chatter message edit and delete allows you to make the edits in the messages as well as internal notes of a record. Moreover, you can also delete the messages as well as internal logs and keep the record of it.

All changes in message/log note will be reflected without page refresh.

Odoo has offered a great functionality of chatter below any form. It allows you to log any internal note or a message to all followers. Odoo Chatter helps you to keep the followers updated regarding the information you want to convey.

However, being human there are many chances to make typos. By default in Odoo, once messages are sent or internal logs are posted then they cannot be deleted or edited.

Edit even after Sending the Message or Log

Now, once sent you can edit the message/log in the Odoo chatter. Yes, Odoo chatter message edit and delete has made it possible to edit the already sent messages or internal logs. Don't worry! You can keep track of all the edits in the Odoo backend.

Expand the view of the Chatter Communication

You can view the whole conversation of the chatter thread; simply by clicking on the expand icon. It helps you to track the chat easily.

Detailed Features List

Below is the detailed list of Feature for Odoo chatter message edit and delete

  • Edit the messages/internal log in the chatter
  • Track the Edit History of the chatter in Odoo Backend
  • Followers of the record will receive a mail for each edit
  • Track the Messages of the chatter in the expanded view
  • Once a user will edit a message/log note then it will be visible in chatter with edit date time
